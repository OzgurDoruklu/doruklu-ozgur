# Proje Bağlamı (Project Context)

**Domain:** `ozgur.doruklu.com`
**Platform:** GitHub Pages
**DNS Yönetimi:** GoDaddy (CNAME kaydı olarak yapılandırıldı)

## Tasarım Dili ve Mevcut Durum
Bu sayfa Özgür Doruklu için tasarlandı. Zümrüt Yeşili (#059669, #047857) tonlarında geliştirilmiştir. Tıpkı ana domainde olduğu gibi modern, dinamik ve karanlık tasarım dili benimsenmiştir. `ozgur@doruklu.com` mailleri bu alan adı kurulumu esnasında Zoho Mail'e (MX kayıtları ile) aktarılarak ücretsiz profesyonel mail hesabı olarak ayarlanmıştır.

## Talimatlar
Bundan sonraki tüm HTML/CSS güncellemeleri yeşil ağırlıklı dark mode temasına uygun olmalı, `index.html` mimarisi korunarak genişletilmelidir.
